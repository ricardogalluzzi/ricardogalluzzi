## 🏆 Projeto em destaque

### Tecnologia de Extração de Hidroxiapatita

Desenvolvimento de um processo tecnológico patenteado (**BR102021001251 B1**) para extração assistida por ultrassom de hidroxiapatita a partir de resíduos de escamas de peixe.

O repositório reúne documentação técnica, informações científicas, dados abertos e elementos de reprodutibilidade relacionados ao desenvolvimento da tecnologia.
